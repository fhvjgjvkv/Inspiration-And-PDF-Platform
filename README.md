# ✨ منصة الإلهام وتحويل الملفات

نظام برمجى متكامل يجمع بين قسم الإلهام والذكر وقسم معالجة وتحويل المستندات الرقمية إلى ملفات PDF احترافية.

## 👩‍💻 تم التطوير بواسطة
**شيماء علي عبد الحسين**

## 💡 ميزات النظام
- **قسم الإلهام والذكر:** - يعرض كلمة تحفيزية تتغير تلقائياً أو عند الطلب.
  - يعرض آية الكرسي وسورة الناس بشكل دائم.
  - يختار آيتين إضافيتين عشوائياً من بنك يضم 48 آية قرآنية.
- **قسم تحويل الملفات:** - يتيح كتابة النصوص ورفع الصور (JPG, PNG).
  - يقوم بدمجها وتوليد ملف PDF عالي الجودة مع إمكانية التنزيل الفوري.

## 🛠️ تقنيات المشروع (Tech Stack)
- **لغة البرمجة:** Python
- **واجهة المستخدم:** Streamlit
- **توليد الـ PDF:** ReportLab & Pillow

## 🚀 طريقة التشغيل محلياً
لتشغيل المشروع على جهازك، تأكد من تثبيت Python ثم اتبع الخطوات التالية:

1. قم بتحميل المستودع:
`git clone <رابط_المستودع>`

2. قم بتثبيت المكتبات المطلوبة:
`pip install -r requirements.txt`

3. قم بتشغيل النظام:
`streamlit run app.py`
